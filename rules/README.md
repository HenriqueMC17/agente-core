Abaixo está o mapeamento dos recursos essenciais deste diretório:

- `master-rules.md` — Constituição central e regras mestras do repositório
- `api-design-rules.md` — Diretivas estritas de design e contratos de APIs
- `backend-rules.md` — Regras de engenharia de backend e qualidade de APIs

- `frontend-rules.md` — Regras de UX/UI, frameworks visuais e design dinâmico
- `fullstack-rules.md` — Padrões integrados de arquitetura fullstack corporativa
