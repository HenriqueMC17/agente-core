# 👑 [MCP-INTEGRATIONS] — Integrações de Model Context Protocol, Conectores & Adapters

Ponto de controle das integrações MCP. Permite a conexão contínua de modelos de linguagem a ferramentas de desenvolvimento locais e nuvem.

---

## 📂 Árvore Semântica Estrutural

Abaixo está o mapeamento dos recursos essenciais deste diretório:

- `mcp_config.json` — Configuração global dos servidores e ferramentas MCP ativos
- `adapters/` — Adaptadores personalizados de comunicação
- `connectors/` — Conectores específicos para bancos de dados e APIs
- `providers/` — Provedores de contexto estruturado
- `runtime/` — Gerenciamento de execução local de ferramentas MCP

---
