# 🛡️ Workspace: Agente Especialista DevOps e SecOps

Ambiente isolado focado no ciclo de vida de operações de desenvolvimento e blindagem. Restringe a visibilidade da IA para garantir a não-poluição inter-departamental.

## 🛠 Como trabalhar neste nível?
Abra esta pasta diretamente na IDE (`subagents/agent-devops-sec`). A IA se adaptará como Especialista em Nuvem e Segurança.

## 📥 Busca de Conhecimento Dinâmico (fetch_skill)
Importe scripts avançados e padrões de nuvem via script interno:

```bash
python ../../12-internal-tools/scripts/fetch_skill.py aws-eks
```
O conteúdo irá diretamente para a raiz local `/.local-skills`, enriquecendo a precisão sem herdar todo o framework.
