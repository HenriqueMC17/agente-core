Workspace super focado em Quality Assurance, Testes E2E, Integração e Manuais Técnicos. Isola lógicas confusas de implementação das exigências limpas da Qualidade para garantir testabilidade.

Abra a pasta diretamente na IDE (`subagents/agent-qa-docs`).
